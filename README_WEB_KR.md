# 웹사이트 사용법 (깃허브 몰라도 가능)

> 완전 초보용 1페이지 가이드는 `QUICKSTART_NONTECH_KR.md`를 먼저 보세요.

## 먼저 이해하기
- 맞아요. **파일이 로컬 PC에 있어야 실행**됩니다.
- 핵심 파일: `index.html`, `app.js`, `styles.css`.

## 실행
- Mac: `start_web.command` 더블클릭
- Windows: `start_web.bat` 더블클릭
- 수동: `python run_studio.py`

## 새 기능(이번 반영)
1. **비율용 심도맵 조절**
   - `Depth 감마`, `Depth 단계 수`, `배경 암부 강화`, `반전 심도`
   - 인물 체형/비율 고정 레퍼런스용으로 심도 형태를 조정 가능
2. **시트 입력 순서 규격화**
   - 이미지1 캐릭터 / 이미지2 배경 / 이미지3 도구 / 이미지4 원본프레임 / 이미지5 심도맵
3. **시퀀스 즉시 프롬프트화**
   - 프레임 카드에서 `원본→프롬프트`
   - 상단에서 `전체 프레임 즉시 프롬프트화`

4. **기획안 파싱 + 추가컷 생성**
   - 3번 섹션에 TB-XXX-YY/NT-XXX-YY 기획안 붙여넣기
   - `기획안 파싱` -> `기획컷 프롬프트 생성`

## 사용 순서
1. 영상 업로드
2. 컷 분할 시작
3. 필요하면 심도 파라미터 조절 후 `심도 다시 생성`
4. 프레임 카드에서 `원본→프롬프트` 또는 `전체 프레임 즉시 프롬프트화`
5. 필요 시 `컷 CSV 내보내기`

## 주의
- 심도맵은 AI 추정 심도가 아니라 **비율/포즈 레퍼런스용 형태화 맵**입니다.
- 결과 품질을 위해 최종 생성 전 프롬프트를 1회 검수하세요.


- 상세 기획/전환 의견 문서는 `FLOW_CONVERSION_STRATEGY_KR.md` 참고.
