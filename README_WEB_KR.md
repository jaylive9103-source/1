# 웹사이트 사용법 (깃허브 몰라도 가능)

## 먼저 이해하기
- 맞아요. **파일이 로컬 PC에 있어야 실행**됩니다.
- 현재 깃(저장소)에 있는 파일을 내 컴퓨터에 내려받아 실행해야 해요.
- 핵심 파일: `index.html`, `app.js`, `styles.css`.

## 1) 가장 쉬운 실행 (버튼처럼 1번 클릭)
### Mac
1. 폴더에서 `start_web.command` 더블클릭
2. 브라우저가 자동으로 열리면서 실행됨

### Windows
1. 폴더에서 `start_web.bat` 더블클릭
2. 브라우저 주소창에 표시된 주소(`http://127.0.0.1:8080/index.html` 같은 형태)로 접속

## 2) 터미널 실행 (수동)
1. 터미널에서 프로젝트 폴더로 이동
```bash
cd /workspace/1
```
2. 실행
```bash
python run_studio.py
```
3. 브라우저에서 자동 오픈 또는 아래 주소 직접 접속
- `http://127.0.0.1:8080/index.html` (포트가 바뀔 수 있음)

## 3) 사용 순서
1. `레퍼런스 영상` 업로드
2. `최대 분석 길이`, `샘플 간격`, `민감도` 설정
3. `컷 분할 시작` 클릭
4. 추출된 카드에서
   - 좌측: 원본 첫 프레임
   - 우측: 심도(그레이) 프리뷰
   - `원본 JPG`/`심도 JPG` 다운로드 가능
5. 프롬프트 영역에서
   - `현재 프레임 프롬프트 생성`: 선택한 프레임 1개
   - `전체 프레임 일괄 생성`: 추출된 프레임 전부
6. `컷 CSV 내보내기`로 샷 매핑 CSV 다운로드

## 4) 추천 설정
- 시작값: `샘플 간격 1.0초`, `민감도 0.18`
- 컷이 너무 적으면: 민감도 낮추기 (예: 0.12)
- 컷이 너무 많으면: 민감도 올리기 (예: 0.25)
- 긴 영상은 먼저 `최대 분석 길이`를 60~120초로 테스트

## 5) 주의사항
- 이 도구는 로컬 브라우저에서 동작한다(서버 업로드 없음).
- 컷 감지는 샘플링 기반이라 완전 정확하지 않을 수 있다.
- 정확한 방송용 결과를 위해 추출 컷을 사람이 최종 검수하는 것을 권장.
