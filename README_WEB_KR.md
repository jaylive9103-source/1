# 웹사이트 사용법 (깃허브 몰라도 가능)

## 실행 방법
1. 이 폴더(`/workspace/1`)에서 아래 명령 실행:
```bash
python -m http.server 8080
```
2. 브라우저에서 `http://localhost:8080` 접속.
3. `index.html` 화면에서 영상 업로드 후 `컷 분할 시작` 클릭.

## 내가 할 수 있는 일
- 영상 업로드
- 컷 전환 후보를 자동 감지해서 첫 프레임 추출
- 추출 프레임 다운로드
- 장면 타입 선택 후 나노바나나용 프롬프트 자동 생성/복사

## 주의
- 이 도구는 브라우저에서 로컬로 동작한다(서버 업로드 없음).
- 컷 감지는 샘플링 기반이라 완전 정확하지 않을 수 있다. `샘플 간격`, `민감도` 값을 조절해 보정한다.
- 긴 영상은 `최대 분석 길이`를 작게 시작한 뒤 점진적으로 늘리는 것을 권장.
