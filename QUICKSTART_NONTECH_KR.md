# 진짜 초보용: 다운로드부터 실행까지 (깃허브 몰라도 됨)

## A. 먼저 이것만 기억
- 이 프로그램은 **웹사이트 파일**이라서,
- 1) 파일을 내 PC로 **다운로드**하고
- 2) 실행 파일(**더블클릭**)을 눌러야 켜진다.

---

## B. 다운로드 방법 (가장 쉬움: ZIP)

### 1) GitHub 페이지 열기
- 이 프로젝트가 있는 GitHub 페이지를 연다.

### 2) 초록색 `Code` 버튼 클릭
- 메뉴에서 `Download ZIP` 클릭

### 3) ZIP 압축 풀기
- 다운로드한 `something.zip`을 더블클릭해서 압축 해제
- 폴더 이름 예시: `insert-prompt-studio-main`

---

## C. 실행 방법 (진짜 버튼 방식)

### Mac
1. 압축 푼 폴더 열기
2. `start_web.command` **더블클릭**
3. 브라우저가 자동으로 열리면 성공

### Windows
1. 압축 푼 폴더 열기
2. `start_web.bat` **더블클릭**
3. 검은 창이 열리고, 브라우저 주소창에 아래 주소 입력
   - `http://127.0.0.1:8080/index.html`

---

## D. 실행이 안 될 때 (딱 2개만 체크)

### 1) Python이 없는 경우
- 메시지에 `python not found` 비슷한 문구가 뜨면 Python 설치 필요
- 설치 후 다시 `start_web.command` 또는 `start_web.bat` 더블클릭

### 2) 포트 충돌
- 주소가 안 열리면 아래 순서로 시도:
  - `http://127.0.0.1:8081/index.html`
  - `http://127.0.0.1:8090/index.html`

---

## E. 앱 안에서 실제 작업 순서
1. `레퍼런스 영상` 업로드
2. `컷 분할 시작` 클릭
3. 나온 컷 중 원하는 프레임 선택
4. `현재 프레임 프롬프트 생성` 또는 `전체 프레임 일괄 생성`
5. 필요하면 `컷 CSV 내보내기`
